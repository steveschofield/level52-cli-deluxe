### 1. MS17‑010 SMBv1 Remote Code Execution  
**Severity:** CRITICAL  
**Affected Component/Service:** SMBv1 service on host **192.168.1.232**  
**Technical Description:**  
The target machine’s SMBv1 implementation is vulnerable to the MS17‑010 flaw (CVE‑2017‑0143). An unauthenticated attacker can send a crafted SMB packet that triggers a buffer overflow in the server’s handling of the `SMB_COM_TRANSACTION2` command, allowing arbitrary code execution with SYSTEM privileges. The vulnerability is triggered by the `SMB_COM_TRANSACTION2` request containing a malicious `SMB_COM_TRANSACTION2` structure that overflows the server’s stack.  

**Evidence & Proof of Concept:**  
```xml
<script id="smb-vuln-ms17-010" output="
  VULNERABLE:
  Remote Code Execution vulnerability in Microsoft SMBv1 servers (ms17-010)
    State: VULNERABLE
    IDs:  CVE:CVE-2017-0143"/>
```
Evidence was captured in two Nmap XML reports (`nmap_20260317_195636_713151.xml`, `nmap_20260317_200047_116558.xml`) and the session JSON (`session_20260317_195441.json`).  

**Impact Analysis:**  
- **Confidentiality:** Full system compromise; attacker can read, modify, or delete any data.  
- **Integrity:** Arbitrary code execution allows modification of system files, registry, and installed software.  
- **Availability:** Potential for denial‑of‑service via crash or resource exhaustion.  
- **Privilege Escalation:** Execution occurs with SYSTEM privileges, enabling persistence, lateral movement, and data exfiltration.

**Detailed Remediation Steps**  
1. **Patch the SMBv1 service** – Apply the latest cumulative update for Windows Server 2008 R2 SP1 (KB4012212 or later).

### 3. Open DNS Recursion  
**Severity:** MEDIUM  
**Affected Component/Service:** DNS server on host **192.168.1.232**  
**Technical Description:**  
The DNS server is configured to allow recursive queries from any client. Recursive resolution can be abused for DNS amplification attacks, facilitating large‑scale DDoS campaigns.
