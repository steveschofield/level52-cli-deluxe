### 1. MS17‑010 SMBv1 Remote Code Execution Severity: CRITICAL Affected Component/Service: SMBv1 service on host 192.168.1.232 Technical Description: The target machine’s SMBv1 implementation is vulnerable to the MS17‑010 flaw (CVE‑2017‑0143). An unauthenticated attacker can send a crafted SMB packet that triggers a buffer overflow in the server’s handling of the `SMB_COM_TRANSACTION2` command, allowing arbitrary code execution with SYSTEM privileges. The vulnerability is triggered by the `SMB_COM_TRANSACTION2` request containing a malicious `SMB_COM_TRANSACTION2` structure that overflows the server’s stack.
