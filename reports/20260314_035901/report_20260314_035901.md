*Source: schemathesis scan output (artifact: session_20260314_035901.json)*

**Impact Analysis:**  
This vulnerability allows attackers to bypass authentication entirely for critical business logic. An attacker can create fraudulent orders, manipulate inventory, or inject malicious data into the database without any credentials.
